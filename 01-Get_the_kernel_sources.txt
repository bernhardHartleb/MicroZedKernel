    Get the sources
    ==============

- Branches information
    
    http://www.wiki.xilinx.com/Using+Git

- Build the linux kernel
	
	1) fetch sources: 

		~$ git clone git://github.com/Xilinx/linux-xlnx.git
        ~$ cd linux-xlnx
            
            Check out and switch to the branch:

				$ git checkout -b xilinx-v2014.3
